# Sample opengl

Drawing Houses with opengl. 

## Screenshot

![opengl](https://github.com/rezaduty/sample-opengl/blob/master/houses.png?raw=true)

## Building

linux: sudo g++ houses.cpp -lGL -lGLU -lGLEW -lglut -o h

windows: please rename `<GL/glut.h>` to `<GL/GLUT.h>`
